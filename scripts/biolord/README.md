## biolord reproducibility 
1. Download the sci-Plex 3 data to from [figshare](https://figshare.com/ndownloader/files/39324305) to `./data/perturbation/`
2. Go to current folder:
    ```{bash}
   cd /biolord_reproducibility/scripts/biolord_scripts
   ```
3. Run sweep:
    ```{bash}
       python base_experiment_sciplex3.py
   ```
